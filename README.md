# URL Shortener

## Overview

A simple Python implementation of a URL shortening system.

---

## Features

* Generate short codes
* Map short codes to original URLs
* Retrieve original URL

---

## Files

* `shortener.py` → generates short code
* `url_map.py` → stores and retrieves URLs

---

## Usage

```bash id="w9p4zr"
python url_map.py
```

---

## Example Output

```id="k2m7qx"
Short code: aB3xYz
Original URL: https://example.com
```
